CC Discord

Discord agents powered by Claude Code

One autonomous Claude Code agent per channel. Messages flow through a relay, Claude processes them, replies go back to Discord. No API key needed — runs on your existing Claude subscription.

  • Anthropic was banning people using their subscriptions with OpenClaw
  • I wanted to use my Claude subscription with an agent — legitimately

Core Features

📡 Per-Channel Agents

  • One Claude agent per channel, plus per-thread agents
  • Each agent has independent conversation context
  • Messages stored in SQLite, delivered via hooks
  • Automatic catch-up of missed messages on restart

⚡ Send Steering

  • If new messages arrive while Claude is composing, the reply is blocked
  • Claude is forced to revise to address all messages
  • No stale or out-of-context responses
  • Ensures every message is acknowledged

🔎 Live Trace Threads

  • Auto-created thread per channel showing live agent activity
  • Watch tool calls, status changes in real time
  • Transparent — users see what the agent is doing
  • Keeps main channel clean, traces in their own thread

Features

💻 Thread Support

  • Threads auto-discovered — agents spawn within seconds
  • Each thread gets its own independent agent
  • Thread context tagged so agents know where they are
  • Typing indicators and replies work inside threads

🔒 Safety + Reliability

  • Bash safety guard blocks risky background operations
  • Stuck agent detection — auto-restart after 15min stale
  • Typing indicators with configurable fallback messages
  • Busy notifications when Claude is already working

📎 Attachments + Memory

  • Discord attachments downloaded and delivered to Claude as files
  • Auto-cleanup after Claude reads them (configurable TTL)
  • Cross-session memory backed by SQLite
  • Relevant prior turns included as context on each message
