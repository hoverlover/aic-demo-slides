Features

> Real Mailboxes

  • Every agent gets a real @chela.email address
  • Full inbound and outbound delivery via Postmark
  • Dedicated Postmark outbound server per account
  • No shared IP reputation — your agent's sending is isolated

> REST + MCP

  • Full REST API for send, receive, search, and manage
  • Native Model Context Protocol (MCP) integration
  • Works with any agent harness: OpenClaw, Hermes Agent, Claude Desktop, PicoClaw, NanoClaw, and others

> Agentic Payments

  • x402 payments with USDC on Base
  • Automated crypto wallet subscriptions or pay-as-you-go credit pack options
  • Stripe checkout available for HITL credit card payments (boring)

Spam + Security

Quarantine & Protection

  • Every inbound email is analyzed and spam-scored
  • Suspicious messages are auto-quarantined — body and attachments withheld
  • Agent is notified and can inspect the quarantined content before deciding
  • Release from quarantine with a single API call once reviewed
  • Detects prompt injection attempts embedded in email body

Features

> Quota-Aware

  • Every send response includes remaining outbound quota
  • Low-balance warnings returned inline so agents can react
  • XMTP wallet notifications for low balance and dunning events
  • Agents can check quota status before deciding to send

> Attachments

  • Served via time-limited signed download URLs
  • No public access — only the owning agent can retrieve
  • File metadata (name, size, content type) included in email records
  • Agent retrieves attachments on demand through the API
