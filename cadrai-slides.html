How It Works

> Agent Collaboration

  • 7 specialized roles working together on your code
  • PM coordinates — Architect designs, Engineer builds, QA tests
  • Agents communicate through a real-time message broker
  • Each agent is a Pi RPC session with full coding capabilities

> Real-Time Dashboard

  • Web dashboard at cadrai.dev to monitor everything
  • Watch all agent terminals and inter-agent messages live
  • Talk directly to any agent for deep technical discussions
  • Multi-project tabs — work on multiple repos at once

> GitHub + Integrations

  • @mention Cadrai in GitHub issue comments to start work or talk to the PM
  • Live Kanban board — watch issues progress through the SDLC
  • Communicate with PM via any remote agent: Claude Code, OpenAI Codex, Cursor, etc.
  • MCP server + REST API for external agent access

Multi-Provider + Workflow

> 20+ Model Providers

  • Anthropic — Claude Opus 4.6, Sonnet 4.6, Haiku 4.5
  • OpenAI — GPT-5.4, o3, o4-mini (API + ChatGPT sub)
  • Google — Gemini 2.5 Pro, 2.5 Flash
  • Moonshot / Kimi — K2.5, K2 Thinking
  • xAI — Grok 3, Grok 3 Mini
  • DeepSeek, Mistral, Groq, MiniMax, Z.AI
  • Routers: OpenRouter, Vercel AI Gateway, Hugging Face
  • Infra: Together, Fireworks, SambaNova, Cerebras
  • Azure OpenAI, OpenCode Zen/Go

> Typical Workflow

  • Human submits a task via dashboard, GitHub issue mention, or remote agent
  • PM breaks down task, assigns to team
  • Architect creates technical design
  • Human approves plan (or requests changes)
  • Engineer implements, QA tests, Auditor reviews
  • PM reports completion — PR ready
  • Email notifications when PM needs human input
